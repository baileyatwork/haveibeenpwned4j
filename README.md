# haveibeenpwned4j

A dependency free Java library for Troy Hunt's ';-- Have I Been Pwned (v3).

## Usage

1. Include the Maven dependency in your pom.xml

```xml
<dependency>
  <groupId>de.martinspielmann.haveibeenpwned4j</groupId>
  <artifactId>haveibeenpwned4j</artifactId>
  <version>1.0.0</version>
</dependency>
```
